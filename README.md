# SEO-Analytics-Project
End-to-end SEO Analytics project with SQL data cleaning, Power BI dashboards, and predictive modeling

## Overview
This project analyzes SEO keyword performance using SQL, Excel, and Power BI.

## Tools Used
- SQL
- Excel
- Power BI

## Objectives
- Analyze keyword search volume
- Identify high-traffic keywords
- Evaluate CPC trends
- Build interactive dashboards

## Key Insights
- Top keywords generated most traffic.
- CPC varied significantly across categories.
- Search volume strongly influenced traffic.

- Dataset
- SQL Queries
